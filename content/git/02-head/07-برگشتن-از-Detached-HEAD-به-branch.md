# 7. برگشتن از Detached HEAD به branch

برای برگشتن به branch اصلی:

```bash
git switch main
```

یا اگر branch اصلی پروژه `master` است:

```bash
git switch master
```

با دستور قدیمی‌تر:

```bash
git checkout main
```
