# 12. دستورهای روزمره

برای کار روزمره همین‌ها کافی است:

```bash
git status
git show HEAD
git show HEAD~1
git log --oneline
git reset --soft HEAD~1
git reset HEAD~1
git reset --hard HEAD~1
```

مهم‌ترین نکته:

> `HEAD` یعنی نقطه‌ای که الان Git وضعیت فعلی پروژه را از آن حساب می‌کند.
