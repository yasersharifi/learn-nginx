# 2. دیدن وضعیت HEAD

برای دیدن وضعیت فعلی repository:

```bash
git status
```

اگر ببینی:

```txt
On branch main
```

یعنی `HEAD` الان روی branch `main` است.

برای دیدن آخرین commit که `HEAD` به آن اشاره دارد:

```bash
git show HEAD
```
