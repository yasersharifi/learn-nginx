# 6. سناریو: فقط fast-forward مجاز باشد

### جواب

```bash
git pull --ff-only
```

فقط وقتی fast-forward ممکن است pull می‌کند؛ اگر divergent باشند خطا می‌دهد. برای جلوگیری از merge ناخواسته مفید است.
