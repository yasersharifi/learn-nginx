# 2. سناریو: انتخاب بهترین update

### جواب

بهترین حالت اینجا **fast-forward** است.

چون local هیچ commit اضافه‌ای ندارد و فقط باید خودش را به remote برساند.

```bash
git pull
```

یا برای سخت‌گیرانه‌تر بودن:

```bash
git pull --ff-only
```
