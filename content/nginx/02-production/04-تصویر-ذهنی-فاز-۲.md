# تصویر ذهنی فاز ۲

در production، Nginx فقط یک واسطه ساده نیست. Nginx جلوی سیستم می‌ایستد و باید جلوی خیلی از مشکلات را بگیرد.

```text
Internet
  ↓
Nginx
  ├── TLS termination
  ├── Request routing
  ├── Rate limiting
  ├── Static serving
  ├── Caching
  ├── Compression
  ├── Logging
  ├── Failure handling
  └── Reverse proxy
        ↓
      Apps / APIs / Services
```

برای هر نقش بالا باید بدانی چه کانفیگی می‌نویسی.
