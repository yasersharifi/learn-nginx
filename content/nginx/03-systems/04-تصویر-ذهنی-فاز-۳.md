# تصویر ذهنی فاز ۳

Nginx در ظاهر یک config file دارد، اما در عمل روی این لایه‌ها سوار است:

```text id="fy4a31"
HTTP / HTTPS
  ↓
TLS
  ↓
TCP
  ↓
Socket API
  ↓
Linux Kernel
  ↓
NIC / Network
```

از سمت process هم:

```text id="y3g83o"
Nginx Master Process
  ↓
Worker Processes
  ↓
Event Loop
  ↓
epoll
  ↓
Non-blocking sockets
  ↓
Client and upstream connections
```

فاز ۳ یعنی فهمیدن این دو تصویر.
