# 7. پیدا کردن فایل‌ها

### پیدا کردن فایل با نام

```bash
find . -name "package.json"
```

### پیدا کردن فایل‌های env

```bash
find . -name ".env*"
```

### پیدا کردن فایل‌های بزرگ

```bash
find . -type f -size +100M
```

### پیدا کردن فایل‌های تغییرکرده در ۲۴ ساعت گذشته

```bash
find . -type f -mtime -1
```
