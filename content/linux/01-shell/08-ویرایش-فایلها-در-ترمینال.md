# 8. ویرایش فایل‌ها در ترمینال

### nano ساده‌تر است

```bash
nano .env
```

کلیدهای مهم nano:

| کلید | کاربرد |
|---|---|
| `Ctrl + O` | ذخیره |
| `Enter` | تایید ذخیره |
| `Ctrl + X` | خروج |

### vim حرفه‌ای‌تر است

```bash
vim nginx.conf
```

حداقل چیزهایی که باید از vim بدانید:

| دستور | کاربرد |
|---|---|
| `i` | ورود به حالت ویرایش |
| `Esc` | خروج از حالت ویرایش |
| `:w` | ذخیره |
| `:q` | خروج |
| `:wq` | ذخیره و خروج |
| `:q!` | خروج بدون ذخیره |
