# 14. متغیرهای محیطی

### دیدن envهای فعلی

```bash
env
```

### جستجوی یک env

```bash
env | grep NODE_ENV
```

### تعریف موقت env

```bash
export NODE_ENV=production
```

### اجرای دستور با env خاص

```bash
NODE_ENV=production npm start
```

### فایل `.env`

نمونه:

```env
NODE_ENV=production
PORT=3000
DATABASE_URL=postgresql://user:pass@localhost:5432/app
JWT_SECRET=super-secret
```

> فایل `.env` را معمولاً نباید commit کنید.
