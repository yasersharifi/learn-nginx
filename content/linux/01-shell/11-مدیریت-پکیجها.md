# 11. مدیریت پکیج‌ها

روی Ubuntu/Debian معمولاً از `apt` استفاده می‌شود.

### آپدیت لیست پکیج‌ها

```bash
sudo apt update
```

### آپگرید پکیج‌ها

```bash
sudo apt upgrade
```

### نصب پکیج

```bash
sudo apt install nginx
```

### حذف پکیج

```bash
sudo apt remove nginx
```

### دیدن نسخه برنامه

```bash
node -v
npm -v
git --version
nginx -v
```
