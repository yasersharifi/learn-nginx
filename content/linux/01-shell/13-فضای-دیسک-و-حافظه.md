# 13. فضای دیسک و حافظه

### دیدن فضای دیسک

```bash
df -h
```

### دیدن حجم پوشه‌ها

```bash
du -sh *
```

### پیدا کردن پوشه‌های سنگین

```bash
du -h --max-depth=1 | sort -h
```

### دیدن RAM

```bash
free -h
```

مثال کاربردی:

```bash
du -sh node_modules .next dist build logs
```
