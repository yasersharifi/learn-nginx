# 5. خواندن فایل‌ها

### نمایش کل فایل

```bash
cat package.json
```

### خواندن صفحه‌به‌صفحه

```bash
less app.log
```

در `less`:

| کلید | کاربرد |
|---|---|
| `q` | خروج |
| `/text` | جستجو |
| `n` | نتیجه بعدی |
| `Shift + G` | رفتن به آخر فایل |

### دیدن ابتدای فایل

```bash
head app.log
```

مثال با تعداد خط:

```bash
head -n 50 app.log
```

### دیدن انتهای فایل

```bash
tail app.log
```

### دنبال کردن لاگ زنده

```bash
tail -f app.log
```

برای production خیلی مهم است:

```bash
tail -f /var/log/nginx/error.log
```
