# 2. جابه‌جایی در ترمینال

### دیدن مسیر فعلی

```bash
pwd
```

مثال خروجی:

```bash
/home/ubuntu/app
```

### رفتن به یک مسیر

```bash
cd /path/to/directory
```

مثال:

```bash
cd /var/www/my-app
```

### برگشتن به مسیر خانه

```bash
cd ~
```

### برگشتن یک پوشه عقب‌تر

```bash
cd ..
```

### برگشتن به مسیر قبلی

```bash
cd -
```
