# 10. sudo و کاربر root

بعضی کارها نیاز به دسترسی admin دارند:

```bash
sudo systemctl restart nginx
```

ورود به shell با دسترسی root:

```bash
sudo -i
```

برگشت:

```bash
exit
```

> با root فقط وقتی کار کنید که لازم است. اشتباه در root گران تمام می‌شود.
