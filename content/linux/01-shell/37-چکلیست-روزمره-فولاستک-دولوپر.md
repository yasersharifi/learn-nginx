# 37. چک‌لیست روزمره فول‌استک دولوپر

### روی پروژه local/server

```bash
pwd
ls -la
git status
git pull
pnpm install
pnpm build
pnpm start
```

### روی سرور

```bash
df -h
free -h
sudo systemctl status nginx
sudo nginx -t
pm2 list
pm2 logs my-app
curl -I http://localhost:3000
curl -I https://example.com
```

### برای Docker

```bash
docker ps
docker compose ps
docker compose logs -f
docker compose up -d --build
```
