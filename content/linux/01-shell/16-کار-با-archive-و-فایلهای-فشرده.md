# 16. کار با archive و فایل‌های فشرده

### فشرده‌سازی با tar.gz

```bash
tar -czvf app.tar.gz app/
```

### باز کردن tar.gz

```bash
tar -xzvf app.tar.gz
```

### zip کردن

```bash
zip -r app.zip app/
```

### unzip کردن

```bash
unzip app.zip
```
