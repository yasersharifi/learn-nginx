# 24. Git در لینوکس

### وضعیت repo

```bash
git status
```

### گرفتن آخرین تغییرات

```bash
git pull
```

### دیدن branchها

```bash
git branch
```

### تغییر branch

```bash
git checkout main
```

یا جدیدتر:

```bash
git switch main
```

### دیدن remoteها

```bash
git remote -v
```

### دیدن لاگ خلاصه

```bash
git log --oneline --graph --decorate --all
```

### stash کردن تغییرات

```bash
git stash
```

### برگرداندن stash

```bash
git stash pop
```
