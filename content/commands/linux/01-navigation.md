# مسیر و پیمایش

| دستور | کاربرد |
| --- | --- |
| `pwd` | مسیر فعلی |
| `cd /path` | رفتن به مسیر |
| `cd ~` | خانه کاربر |
| `cd ..` | یک پوشه بالا |
| `cd -` | مسیر قبلی |
| `ls` | لیست فایل‌ها |
| `ls -la` | لیست با جزئیات و فایل‌های مخفی |
| `ls -lh` | اندازه‌ها به‌صورت خوانا (K/M/G) |
| `tree -L 2` | درخت پوشه (اگر نصب باشد) |

مسیرهای پرتکرار:

```bash
/etc          # کانفیگ سیستم
/var/log      # لاگ‌ها
/home/user    # خانه کاربر
/tmp          # موقت
```
